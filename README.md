# Curso de Lógica de Programação e Algoritmos

Este repositório foi criado para centralizar e organizar todos os meus estudos de lógica de programação, conceitos fundamentais e resolução de algoritmos.

Aqui estão os exercícios práticos focados no desenvolvimento do raciocínio lógico, estruturas condicionais, laços de repetição, vetores e matrizes.

## 🛠️ Tecnologias e Ferramentas

- **Portugol Web Studio** / Visualg (Escrita de pseudocódigos)
- **Git & GitHub** (Controle de versão e documentação)

## 📂 Estrutura do Repositório

Os arquivos estão organizados de forma descritiva dentro das seguintes pastas:

- **exercicios-portugol/**: Pasta principal contendo os algoritmos desenvolvidos em Portugol (`.por`).
  - `calculo_imc.por`: Algoritmo para cálculo de Índice de Massa Corporal com condicionais.
  - `listagemturma_vetores.por`: Exercícios práticos aplicando conceitos de vetores.
